Assignment [[[[file:202101021700-python_operators.org][python operators]]]] are used to assign values to variables:

- === *equals*, for example =x = 5= is the same as =x = 5=
- =+== for example =x +=3= is the same as =x = x + 3=
- =-== for example =x -= 3= is the same as =x = x - 3=
