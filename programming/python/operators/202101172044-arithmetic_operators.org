Arithmetic [[[[file:../202101021700-python_operators.org][python operators]]]] are used with numeric values to perform common
mathematical operations:

| Operator | Name           | Example |
|----------+----------------+---------|
| +        | Addition       | =x + y=   |
| -        | Subtraction    | =x - y=   |
| *        | Multiplication | =x * y=   |
| %        | Modulus        | =x % y=   |
| **       | Exponentiation | =x ** y=  |
| //       | Floor Division | =x // y=  |
